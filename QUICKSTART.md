# nuFit - Quick Start Guide

## Getting Started

You've successfully created the nuFit fitness tracking app! Here's how to run it:

## Run the App

1. **Start the development server:**
   ```bash
   cd nuFit
   npm start
   ```

2. **Choose your platform:**
   - Press `i` for iOS simulator (macOS only)
   - Press `a` for Android emulator
   - Press `w` for web browser
   - Scan QR code with Expo Go app on your phone

## How to Use

### Log a Workout
1. Go to "Workouts" tab
2. Enter active calories burned
3. Enter total calories burned
4. Tap "Add Workout"

### Log Food
1. Go to "Nutrition" tab
2. Enter food name
3. Enter number of servings
4. Enter protein per serving (grams)
5. Tap "Add Food"

### Adjust Protein Goal
1. Go to "Dashboard" tab
2. Scroll to "Daily Protein Goal" card
3. Tap "Change Goal"
4. Enter new goal
5. Tap "Save"

## Data Storage

All your data is automatically saved on your device. It will persist even if you close the app!

## Need Help?

If you encounter any issues:
1. Make sure all dependencies are installed (`npm install`)
2. Clear Metro bundler cache: `npm start -- --reset-cache`
3. Check that you have the correct Node.js version
